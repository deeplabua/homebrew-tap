# DeepLab Homebrew Tap

Homebrew tap for the DeepLab line of tools.

```sh
brew install deeplabua/tap/deepshrink
```

- Remote: `git@github.com:deeplabua/homebrew-tap.git` (public)
- **Formulae here are auto-generated and committed by `cargo-dist`** on each release
  (see `deepshrink`'s `[workspace.metadata.dist]`). Don't hand-edit `Formula/*.rb`.
- The `deepshrink` formula declares `depends_on "ffmpeg"`, so brew installs ffmpeg too.
